Islands

Ahoy navigator! Tell me,
which points are on the
same island?

▪ Given a map and a list of coordinate
pairs, for each pair, find out
whether both coordinates are on the
same island.
▪ In this level, all given coordinates
will be on land.
▪ Islands don’t connect diagonally.

Input
- Size of Map : An integer representing the width and height of the map
- Map :  A paragraph of characters
- N : An integer representing the number of coordinate pairs
- Coordinate Pair (repeated N times) : A pair of coordinates separated by a space. The order is X1,Y1 X2,Y2

Input example :
11
WWWWWWWWWWW
WLWWWWWWWWW
WWWWLLLLWWW
WWWLLLLLWWW
WWWLWWWLLWW
WWLLWLWWLWW
WWLLWWWWWWW
WWWWLWWWLWW
WWLWWWLLWWW
WLLWWWWWWWW
WWWWWWWWWWW
3
4,2 8,5
3,5 4,7
5,5 7,8

Output : 
Result (repeated N times)

Output example :
SAME
DIFFERENT
DIFFERENT
